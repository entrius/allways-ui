# Working agreements

## Commits and PRs

Do not add Claude/AI attribution to any commit message or PR body. That means:

- No `Co-Authored-By: Claude ...` trailer in commit messages.
- No `🤖 Generated with [Claude Code]` line (or any equivalent) in PR descriptions.
- No "Generated by Claude" / "Written with the help of Claude" footers anywhere.

Write commits and PRs as if a human wrote them. The user knows Claude wrote
the code; they don't want it stamped on every artifact.
